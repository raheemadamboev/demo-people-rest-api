# demo-people-rest-api
Java Spring Boot, REST API to list people details. Application is connected to PostgreSQL database. 

<img src="https://i.imgur.com/Id9o5iJ.jpg" alt="spring boot" width="600" height="520" />

**Tech stack:** 

- Java Spring Boot 
- Docker 
- PostgreSQL 
- Maven 
- FlywayDB 
- JDBCTemplate 
- HikariDataSource
